# RA Causal-DAG Motif-Commit Simulator v0.1 Report — May 4, 2026

## Purpose

This workbench operationalizes the compiled `RA_MotifCommitProtocol` vocabulary in Python so that motif-commit behavior can be explored before downstream Lean refinements are added.

## Lean-to-Python correspondence

```text
Lean MotifCandidate / GraphMotifCandidate
  ↔ Python MotifCandidate

Lean CausalSupportCut / GraphSupportCut
  ↔ Python SupportCut = frozenset[node_id]

Lean supportCutOfFiniteHasseFrontier
  ↔ CausalDAG.support_cut_of_finite_hasse_frontier

Lean DAGReadyAt / GraphReadyAt
  ↔ CausalDAG.ready_at

Lean DAGCommitContext.supports / GraphCommitContext.supports
  ↔ CommitContext.supports

Lean incompatible relation
  ↔ CommitContext.incompatible

Lean DAGCommitsAt / GraphCommitsAt
  ↔ MotifCommitProtocol.decision(...).commits

Lean DAGFinalizedAtDepth / GraphFinalizedAtDepth
  ↔ CausalDAG.finalized_at_depth
```

## Semantics

The simulator uses the strict context-gated commit rule:

```text
commits(M, Q, x) iff
  supports(M, Q)
  and ready_at(Q, x)
  and there is no certified-ready M' incompatible with M at x.
```

This means symmetric incompatible alternatives block each other unless a future selector-closure layer narrows the certified candidate set or adds a deterministic RA-native selector. That is intentional: the simulator is modeling the current Lean predicate, not a later selector protocol.

## Demo behavior

The demo grows an append-only finite DAG. For each new site it:

```text
1. selects parent support vertices;
2. forms the down-closed candidate past generated by those supports;
3. extracts the Hasse frontier as the causal support cut;
4. creates a renewal motif;
5. sometimes creates two incompatible alternatives sharing an exclusion domain;
6. evaluates strict motif commits at the new site.
```

## Next simulator extensions

```text
- Instantiate supports with BDG/local-ledger/orientation predicates.
- Add merge-compatible motif composition.
- Add selector-closure mode that narrows conflicting alternatives before commit.
- Add frontier statistics: support-cut width, depth, antichain size, readiness latency.
- Add Monte Carlo sweeps over conflict rate and parent-selection policy.
```
