# Track B.1 bridge note

`RA_MotifNativeGraphCutWitnessExtraction` connects graph/cut-local simulator
witness extraction to the existing Lean graph-orientation closure ladder.

The bridge is deliberately one-way:

```text
graph/cut-local witness evidence
  -> orientation closure sign source
  -> seven-value N1 ledger surface
```

It does not claim that graph/cut witnesses produce orientation-specific rescue.
That claim was explicitly retracted for the current matched-graph pipeline by the
v1.7/v1.8.1 audit chain.
