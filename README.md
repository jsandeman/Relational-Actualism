# Relational Actualism (RA): Formal Verification Repository

Welcome to the formal verification repository for **Relational Actualism (RA)**, a unified process ontology framework that models quantum mechanics, gravity, and the emergence of biological complexity through the discrete growth of a causal Directed Acyclic Graph (DAG).

## 🗂️ Repository Structure
* `src/RA_AQFT/`: Foundational quantum field theory and relativity proofs.
* `src/RA_Complexity/`: Graph-theoretic proofs for biological emergence and agency.
* `data/DFT_Survey/`: B3LYP/6-311+G* density functional theory pipeline and raw output.
* `docs/`: Supplementary mathematical notes and preprint links.
