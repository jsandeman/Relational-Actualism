# Paper IV native rewrite patch v1

## Goal

Replace the current hybrid complexity/life paper with a strictly RA-native extension paper.

Paper IV should no longer be a chain from QCD / GR / quantum information into biology.
It should instead begin from:
- recursive graph coarse-graining
- causal shielding
- recursive closure
- finite assembly depth
- direct biological and cognitive observables

## Highest-priority rewrite

### Abstract
Remove the current chain:
- QCD confinement
- quantum Darwinism
- Unruh-temperature control
- KCB / quantum computing
- Landauer / Maxwell
- Markov blanket as the public name

Replace with something like:

> The fourth paper extends RA into complexity, life, agency, and cognition using only native categories: recursive closure, causal shielding, finite assembly depth, and boundary-maintained organization. The central question is not whether biological complexity mirrors any prior theory object, but whether stable self-maintaining multi-layer causal organization can be predicted directly from DAG + BDG + LLC and compared to observed biochemical and organismic systems.

### Introduction
Replace the sentence claiming that Papers I--III "generate quantum mechanics, the Standard Model, general relativity, and cosmology."

Use instead:

> Papers I--III establish the native discrete bedrock: irreversible graph growth, BDG filtering, ledger conservation, motif dynamics, severance, and macroscopic boundary structure. Paper IV asks what forms of stable recursive organization those same primitives permit.

Remove the five-context `\mu = 1` list if it includes:
- QCD confinement
- galactic rotation curves
- fault-tolerant quantum computing
- any other imported target

The only active use of `\mu = 1` in this paper should be as a **native connectivity / closure threshold** within the coarse-grained graph itself.

## Section 3: Causal Firewall
The term **Causal Firewall** is acceptable as an internal RA name.
What must change are F1 and F2.

### Replace F1
Current version: quantum Darwinism redundancy.

Native replacement:
> **F1 (redundant boundary inscription):** actualization outcomes must be re-written across multiple boundary channels so that local closure does not depend on a single fragile edge or shell fragment.

### Replace F2
Current version: non-relativistic decoherence control / Unruh temperature ratio.

Native replacement:
> **F2 (bounded boundary turnover):** the rate at which boundary conditions are overwritten by ambient activity must remain below the rate at which the interior can re-establish its own closure.

This lets the paper use measured temperature, noise, turnover, and persistence data only as empirical anchors, not as imported theoretical mechanisms.
