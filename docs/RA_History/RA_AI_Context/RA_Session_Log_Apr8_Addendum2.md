## Architecture Change

The 6-paper structure (P0–P5) is replaced by a 5-paper structure (P0 + P1–P4).
P5 (framework summary) is eliminated; its content is absorbed into P0.

### Rationale

P0 ("Biography of the Universe" — narrative overview) and P5 ("Irreversible Events
as Primitive Basis" — framework summary) overlapped heavily.  Both described the full
framework, mapped the constraint web, and catalogued predictions.  The difference was
voice (narrative vs. technical), not content.  A single overview paper is cleaner.
