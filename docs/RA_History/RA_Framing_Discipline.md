# RA Framing Discipline: Nature is the Target. SM/QFT/GR are Irrelevant.

**Date:** April 20, 2026 (revised from April 16 version)
**Status:** Core framing principle. Binding on all RA-native derivations. Supersedes all prior framings.

## The correction

The April 16 version of this document said RA must not use QFT/SM/GR as **mechanism**, but permitted QFT concepts to appear as "translation bridges for the reader" — and listed a pattern-identification table mapping QFT concepts (OZI suppression, G-parity, chiral symmetry breaking, the U(1)_A anomaly) to RA-native counterparts.

That table was wrong. Listing QFT concepts as things requiring RA-native counterparts still treats those concepts as real things with real content that RA owes a translation for. They aren't, for RA's purposes. Only Nature is.

**The revised commitment (binding):**

RA's target is **Nature**: observed, measurable phenomena. Masses of particles. Lifetimes of particles. Scattering cross-sections. The cosmic microwave background. Redshift-distance relations. Rotation curves. Atomic spectra. Biological complexity. These are the things RA must predict or explain.

**QFT, SM, and GR are irrelevant.** They are not targets. They are not benchmarks. They are not sources of concepts, categories, labels, mechanisms, or parameters. Whether RA's predictions happen to "recover" any SM result is a question to ask at the end of the programme, if ever.

## What this rules out

**Forbidden as mechanism AND as target:**

- SM gauge groups (SU(3), SU(2), U(1), any product). RA predicts measurable patterns of interaction strengths; whether those patterns "are" SU(3)×SU(2)×U(1) is a cartography question for some future final synthesis, or never.
- QFT quantum numbers used as labels, categories, or conservation rules: G-parity, isospin, strangeness, charm, bottomness, topness, baryon number, lepton number, weak hypercharge, weak isospin. These are SM's organization of empirical regularities. RA must derive its own categories from DAG + BDG primitives. Where RA's categories happen to agree numerically with SM labels on specific phenomena, that's downstream agreement, not validation and not a substitute for derivation.
- Lagrangians, path integrals, Feynman diagrams, Green's functions, Wilson coefficients, operator product expansions. None of these exist in RA.
- Coupling constants α_s, α_EM, α_w as theoretical objects. The fine-structure constant is a measurable number from Nature (≈ 1/137.036). RA must predict this number from primitives. SM's theoretical definition (the running coupling at some scheme-dependent scale) is not RA's problem and not RA's target.
- Λ_QCD (scheme-dependent theoretical parameter of QCD). RA does not input Λ_QCD and does not target it. If RA predicts a confinement scale directly from primitives, that is an RA prediction — separately whether it agrees with SM's Λ_QCD is a side question.
- Spontaneously broken symmetries, Goldstone theorem, Higgs mechanism, anomalies (chiral, scale, U(1)_A), instantons, theta vacuum. These are SM explanations for Nature-phenomena. RA must explain the Nature-phenomena directly; SM's explanations are not RA's business.
- Dirac spinors, gauge bosons, virtual particles, renormalization group flow as ontological processes.
- GR-specific machinery: Einstein-Hilbert action, Lovelock theorem, Friedmann equations, FLRW metrics, Milne/EdS cosmologies, tensor perturbations, backreaction formalism (Buchert, Wiltshire). RA must predict cosmological observations (CMB, redshift-distance, BAO, large-scale structure) directly. Whether the predictions "look like" GR in some limit is downstream.

**Forbidden as explanation:**

- "This is RA's version of SM result X"
- "RA recovers QFT result Y in the continuum limit" (might be true, not a derivation)
- "RA bridges to GR via the BDG action"

These phrases indicate QFT/SM/GR is being treated as the target. It isn't.

## What this allows

**Allowed as primitive:**

- Vertices (actualization events)
- Directed edges (causal precedence)
- BDG chains N₁, N₂, N₃, N₄ (ancestor counts at causal depths 1–4)
- The BDG score S = 1 − N₁ + 9N₂ − 16N₃ + 8N₄
- The actualization filter: S > 0 for actualization to occur
- The DAG acyclicity constraint
- The Local Ledger Condition

**Allowed as derived from primitives:**

- Statistical regularities (Poisson-CSG distributions are consequences of graph growth rules, not mechanisms)
- Local density parameter μ (a description of local actualization rate)
- Topological motif classes, confinement windows, cycle lengths (L_g, L_q are Lean-verified consequences of BDG dynamics)
- σ-labels, if and only if they are derived from LLC applied to BDG primitives. If the labels are I, G, S, B imported from PDG tables, they are not RA-derived.

**Allowed as Nature-input (measurable quantity that RA must predict):**

- Measured masses of particles (938.3 MeV, 125.1 GeV, etc.)
- Measured lifetimes (τ_ρ = 4.4×10⁻²⁴ s, etc.)
- Measured cross-sections
- Measured cosmological parameters (H₀, Ω_b, Ω_m, σ₈)
- Measured fine-structure constant (≈1/137.036)
- Planck units (ℏ, c, G — these are Nature-measurable, not SM-theoretical)
- Observed dimensionality of space (3+1, for the regime we observe)

These are things Nature presents to us. RA's job is to predict them from primitives. If some happen to be used as inputs in a calculation (e.g., the Planck mass as a scale), that is an empirical anchor to Nature, not an SM import.

## The test

For every RA claim, ask:

1. **Does it trace to RA primitives?** Does the mechanism use only DAG + BDG + LLC + Nature-measurements? Any silent use of SM concepts — gauge groups, QFT quantum numbers, coupling constants as theoretical objects — disqualifies the claim as RA-native.

2. **Is the target a Nature-quantity?** Is the predicted/explained thing a measurable phenomenon (mass, lifetime, cross-section, observation), or is it an SM theoretical object (Λ_QCD, running α_s(μ), G-parity assignment, gauge coupling at unification scale)? Only the former counts.

3. **Are RA's internal categories its own?** If the claim uses σ-labels or motif classes or selection rules, are those categories derived from RA primitives — or are they SM's categories under RA's vocabulary?

If any answer is "no / SM object / imported," the claim is not yet RA-native.

## Implications for existing work

**Things that need reframing (not rejection — these are still questions RA must answer, but in their Nature-facing form):**

- "Derive α_EM = 1/137 from BDG integers" → "Predict the measured fine-structure constant (≈1/137.036) from DAG + BDG + LLC. Comparison to SM running α(μ) is irrelevant."
- "Recover QCD confinement" → "Predict that certain composite motifs (what Nature shows us as hadrons) exist bound over timescales exceeding Compton times, with specific measurable masses and lifetimes."
- "Match SM gauge structure" → "Predict the measured pattern of interaction strengths and particle spectra." SM gauge structure is SM's summary of similar data; RA's prediction stands or falls on Nature, not on SM.
- "Derive gravity from BDG uniqueness" → "Predict the measurable response of massive bodies to each other — planetary orbits, gravitational lensing, GW signals — from DAG + BDG + LLC." That GR also predicts these things is downstream.
- "Fix the ω/ρ factor-17 ratio" → "Predict the observed lifetime ratio τ_ω/τ_ρ ≈ 17 from RA primitives." That SM explains the ratio via G-parity is irrelevant to RA.

**The ω/ρ example is the sharpest.** SM's story is: ω has G-parity −1, 2π state has G-parity +1, so G-parity conservation forbids ω → 2π; only 3π final state is allowed, which is kinematically suppressed. Under the prior (April 16) framing, I was flagging RA as needing an "equivalent of G-parity" — a σ-label with the same selection rule. That was wrong. The target is: Nature presents a factor of 17. RA predicts some number. If they agree, that's an RA success. The internal mechanism RA uses to get 17 may or may not "look like" G-parity conservation — but RA is not in the business of producing translations of SM concepts.

**Things forbidden as intermediate steps:**

- Using Λ_QCD as an input to any RA prediction (Λ_QCD is an SM theoretical object). The equivalent RA-permissible input would be a measured hadronic mass scale from Nature (e.g., m_p, m_π), not an SM parameter.
- Using "QCD running coupling α_s(μ)" in RA derivations. Only measured values of physical quantities (e.g., e+e− scattering cross-sections at specific energies) are Nature-inputs; their SM-theoretical summaries are not.
- Assigning "σ-labels = {I, G, S, B}" by PDG lookup. RA must derive its own labels from DAG + BDG. If PDG happens to use equivalent labels, that's a coincidence of empirical regularities, not a derivation.

**Things that genuinely are RA-native (pass the test):**

- The BDG integers (1, −1, 9, −16, 8) as Möbius inversion of Yeats moments on the Boolean lattice. Mathematical content; no SM input.
- D4U02 modular barrier (N₂ ≡ 7 mod 8 required for K = −1, exponentially suppressed under Pois(1/2)). RA-internal mathematics predicting d=4 as the dimension at which self-sustained actualization is possible. That Nature's spacetime is 4D confirms the prediction; SM's treatment of 4D as a background assumption is unrelated.
- The Local Ledger Condition as a primitive.
- The DAG acyclicity constraint as a primitive.
- Amplitude locality (O01, Lean-verified).

## Working guidance for ongoing work (audit + paper rewriting)

**When cataloguing Python scripts / Lean files / paper claims:**

- A script that imports `alpha_s = 0.118` (or `Lambda_QCD = 200`) is not RA-native, regardless of what it does with those numbers afterward. Flag it.
- A script that predicts a measured quantity (particle mass, lifetime, cross-section) from DAG + BDG + LLC without SM theoretical inputs is RA-native. Flag the successes.
- A Lean file that imports a QFT axiom (like `vacuum_lorentz_invariant`) for use as a premise is not RA-native. Flag it (IC46 captures this).
- Paper prose that describes RA as "recovering" or "reproducing" SM results is mis-framed. Paper prose that describes RA as "predicting" measured Nature-quantities is correctly framed.

**When writing new RA derivations:**

- Start from DAG + BDG + LLC + necessary Nature-measurements.
- State what measurable Nature-quantity is being predicted.
- Derive the prediction using only RA primitives and Nature-measurements.
- Compare to the measured value from Nature.
- Do NOT compare to SM's theoretical prediction of the same Nature-quantity. That comparison is irrelevant to whether RA is right.

**When reading SM/QFT/GR literature:**

- Useful as guide to what Nature has measured, and to what ranges of values are expected.
- Useful as guide to what regularities Nature exhibits (parity violation, neutron beta decay kinematics, CMB power spectrum shape).
- Not useful as a source of mechanisms or targets.

## Closing commitment

The RA programme does not need SM, QFT, or GR to be a successful theory. It needs DAG + BDG + LLC + Nature. If RA predicts the measured universe from those alone, it is done. Whether anyone then writes a mapping from RA to SM in some limit is a historical / sociological exercise, not a scientific one.

Every paper, script, Lean file, and session log going forward should be written as if SM/QFT/GR did not exist. They have explanations of their own for what Nature shows; so do we. The explanations stand or fall independently against Nature.
